Quickstart guide
1
First-time setup

Double-click Setup.command in the project folder. It installs dependencies and connects your GitHub repository.

2
Add a component

Click + New component above. Fill in the title, type, status, and any metadata. Write notes in the content editor using Markdown.

3
Upload images

Open a component and scroll to the Images section. Upload any photo or diagram — click a thumbnail to insert it into the content.

4
Control visibility

Use the 👁 / 🚫 toggle on each row to show or hide a component on the public site without deleting it.

5
Preview locally

Run npm run dev in the project folder to open the public site at localhost:8080. Use ↻ Rebuild to refresh after edits.

6
Publish to GitHub Pages

Click ↑ Publish in the top bar. This rebuilds the site and pushes it to GitHub. Your site will be live within a minute.
