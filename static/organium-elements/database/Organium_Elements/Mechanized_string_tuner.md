# Mechanized string tuner

Added: March 7, 2023 11:28 AM
Status: In progress

![IMG_20230306_171738.jpg](mechanized-string-tuner-1.jpg)

A Skerpla student with interest in a microtonal guitar has fast tracked development of this element. First test with a tiny geared DC motor is promising but the motor stalls before this is a fully useful solution.

Really low geared DC motors incoming off Amazon will hopefully solve the issue.

Running on 6V, 1A
