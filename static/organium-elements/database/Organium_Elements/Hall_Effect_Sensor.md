**Principle:** Resistive (depends)

**Function**: By bending this strip, voltages will vary. This sensor is also known as a bend sensor. By bending the sensor it varies the control voltages, typically from 0 to 5 volts. 

**Voltage**: 3- 5v

**Typical use:** People have used this sensor in data gloves, wearable computing and as part of instruments where you have a hand free or move parts of the instrument. This could also be used as a wind sensor, where a flip is put onto the sensor and the user blows onto it. 

This sensor can easily be tacked onto an existing control interface (keyboard key, slide, levers of various kinds) to get a digital response to a mechanical action.
