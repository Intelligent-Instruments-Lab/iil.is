**Name**: Notochord

**Type**: Software (Machine Learning)

**Principle**: Machine learning model for MIDI performance generation. Similar to text-based models like ChatGPT, but trained on MIDI data for musical output.

**Function**: Generates or harmonizes MIDI music in real-time using machine learning. Prioritizes speed over high fidelity, enabling live performance integration.

**Voltage**: N/A (runs on laptop/computer)

**Typical use**: MIDI generation, MIDI harmonization, live performance augmentation, AI-assisted composition, real-time musical accompaniment.

**Good for**:

- Generating AI-based musical content
- Real-time MIDI processing
- Sculpting random noise into quasi-musical output
- Live performance integration
- Fast response time
- Experimental composition

**Bad for**:

- High-fidelity MIDI reproduction
- Deliberate/precise MIDI processing (use Scramble instead)
- Embedded applications without laptop
- Deterministic musical outcomes
- Traditional composition workflows
